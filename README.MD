# ❌⭕ Telegram Bot: Tic-Tac-Toe for Groups

Телеграм-бот для игры в «Крестики-нолики» внутри групповых чатов. Проект выполнен в рамках учебного курса по программированию студентами 2 курса ФИИТ ЮФУ Лигоровой Марией и Тимофеевым Андреем.

##   Функционал
* **Многопользовательский режим**: Очередь на игру внутри группового чата.
* **Интерфейс**: Интерактивное игровое поле через Inline-клавиатуру Telegram.
* **Статистика**: 
  * Личные достижения (`/my_stats`).
  * Общий рейтинг группы (`/general_stats`).
  * Отслеживание чемпионов и последних победителей.
* **Надежность**: Сохранение всей статистики в YAML-файл в реальном времени.

##  Технологический стек
* **Язык:** Ruby
* **Библиотека:** [telegram-bot-ruby](https://github.com/atkr/telegram-bot-ruby)
* **Тестирование:** RSpec (Moсks & Doubles для имитации API)
* **CI/CD:** GitHub Actions (автоматическая проверка кода при каждом Pull Request)
